A text adventure in pure JS.

## Authoring

Engine/content authoring documentation lives in [storydocs/README.md](./storydocs/README.md).

The two most practical entry points are:

- [storydocs/AUTHORING_GUIDE.md](./storydocs/AUTHORING_GUIDE.md): current engine-facing content reference.
- [storydocs/AUTHORING_TUTORIAL.md](./storydocs/AUTHORING_TUTORIAL.md): step-by-step tutorial for adding a new room feature.

## Testing

Run the reusable test suite from the repo root:

```bash
npm test
```

Available scripts:

```bash
npm run test:unit
npm run test:smoke
```
